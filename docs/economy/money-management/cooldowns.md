# /cooldowns

See when each of your income commands will be ready to use again.

## Description

Shows the remaining cooldown on every recurring command in one place: `/daily`, `/weekly`, `/monthly`, `/hourly`, `/beg`, `/mine`, `/crime`, `/rob`, `/heist`, and weekly scratch tickets. Ready commands are marked with a green toggle; commands still on cooldown show the time remaining.

`/cd` is a shorter alias for the same command.

## Optional Fields

- **user** - Mention another player to view their cooldowns instead of your own.

## Usage

`/cooldowns [user]`
`/cd [user]`

## Examples

`/cooldowns`
`/cd`
`/cooldowns @username`

## Notes

- No cooldown - check as often as you like.
- **Patreon supporters get 35% faster cooldowns** on most commands.
- Heist cooldowns are tracked separately and can be hours long after a successful run.
- Weekly scratches reset every 7 days; the page shows when your next batch unlocks.
