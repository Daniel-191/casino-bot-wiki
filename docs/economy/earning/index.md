# Earning Money

Free money sources - no risk, no gambling required. The recurring rewards (`/daily`, `/hourly`, `/weekly`, `/monthly`) plus `/beg` and `/vote` are how you build a starting bankroll.

## Commands at a glance

| Command | Cooldown | Base reward |
|---|---|---|
| [`/hourly`](hourly.md) | 1 hour | $7.5M + streak bonus |
| [`/daily`](daily.md) | 24 hours | $75M |
| [`/weekly`](weekly.md) | 7 days | $375M |
| [`/monthly`](monthly.md) | 30 days | $2B |
| [`/beg`](beg.md) | 15 seconds | $10K – $100K (random) |
| [`/vote`](vote.md) | 12 hours per site | $50M (scales with streak) |
| [`/vote-milestones`](vote-milestones.md) | - | View streak unlocks |

Use [`/cooldowns`](../money-management/cooldowns.md) to see when each is ready.

## Multipliers stack

Every recurring reward (`/daily`, `/weekly`, `/monthly`, `/hourly`) gets multiplied by **two** independent boosters:

### Vote streak multiplier

| Streak | Multiplier | Tier |
|---|---|---|
| 1+ | 1.25× | Newcomer |
| 7+ | 1.5× | Starting |
| 14+ | 1.75× | Building |
| 21+ | 2× | Growing |
| 30+ | 2.5× | Rising |
| 45+ | 2.75× | Loyal |
| 60+ | 3× | Regular |
| 75+ | 3.5× | Committed |
| 100+ | 4× | Dedicated |
| 125+ | 4.5× | Advanced |
| 150+ | 5× | Veteran |
| 175+ | 5.5× | Expert |
| 200+ | 6× | Master |
| 250+ | 7× | Elite |
| 300+ | 8× | Diamond |
| 356+ | 10× | Legendary |

### Patreon multiplier (DWM)

| Tier | Multiplier |
|---|---|
| Donator (T1) | 2.5× |
| Chill (T2) | 5× |
| Epic (T3) | 10× |
| Super (T4) | 15× |
| Mega (T5) | 20× |
| Ultimate (T6) | 25× |
| Crazy (T7) | 30× |
| Godly (T8) | 40× |

A Godly patron with a Legendary streak earns **400×** the base reward - `/monthly` becomes $800B per claim.

!!! note "What patrons don't get"
    Patreon does not reduce cooldowns on `/daily`, `/weekly`, `/monthly`, or `/hourly` - those stay fixed. Cooldown reductions apply to other commands like `/beg`, `/crime`, `/rob`, etc.
