# Ask for Help

The fastest way to get help with Paradise Bot.

## Order of operations

When you have a question or problem:

1. **Search the wiki.** Use the search bar at the top - chances are someone has already asked.
2. **Check the [FAQ](../faq/customize.md).** Common issues are documented there.
3. **Try `/guide`** - the in-game interactive guide covers most basics.
4. **Open a support ticket** in the Discord server if the above didn't help.

## Discord support server

The fastest way to get a real human to help: [discord.gg/vMcbru7qMV](https://discord.gg/vMcbru7qMV).

- **#general** - community help, casual questions
- **#support-tickets** - open a ticket for account issues, ban appeals, payment problems
- **#bug-reports** - known bugs and reports
- **#suggestions** - feature ideas

## What to include in a help request

Save everyone time by giving:

- **Your Discord ID** (right-click your name → Copy ID).
- **The exact command you ran** (e.g., `/sportsbet 5b`).
- **The exact error message** (screenshot or copy-paste).
- **What you expected to happen** vs. what actually happened.

## Don't bother asking for…

- **Free money handouts** - staff doesn't gift currency.
- **Reset of your stats** - losses are permanent.
- **Custom features just for you** - unique requests rarely make it.
- **Refund of game losses** - by design, gambling games can lose money.
