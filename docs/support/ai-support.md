# AI Support

Paradise Bot does not currently have a built-in AI support feature. This page is reserved for future plans.

## What you can do today

| Need | Where to go |
|---|---|
| **Search the wiki** | The search bar at the top of any wiki page |
| **Open a support ticket** | [Discord support server](https://discord.gg/vMcbru7qMV) |
| **Ask the community** | [Discord support server](https://discord.gg/vMcbru7qMV) |
| **Report a bug** | [`/bugreport`](report-bugs.md) (built-in command) |
| **Suggest a feature** | [`/suggest`](suggestions.md) (built-in command) |
| **Read the rules** | [Rules](../general/rules.md) |
| **Check the FAQ** | [FAQ section](../faq/customize.md) |

## If AI support is added

If a chat-based AI support feature is added in the future, this page will document:

- How to invoke it.
- What kinds of questions it can handle.
- When to escalate to human staff.

For now, please use the human-staffed Discord server for help.
