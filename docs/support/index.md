## Reporting Bugs

If something is broken, use `/bug <description>` in any server where Paradise Bot is active. Reports go directly to the developer.

## Suggesting Features

Have an idea? Use `/suggest <idea>` - every suggestion gets read.

## Rules

- One account per person
- No automation or botting
- No exploiting bugs - report them instead
- Be respectful in shared games

Bans for rule-breaking are usually permanent. See the [FAQ](faq.md) for more.
