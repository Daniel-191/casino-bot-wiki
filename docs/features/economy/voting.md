# Voting & Rewards

Paradise Bot's voting system provides daily rewards for supporting the bot on listing websites, with streak bonuses and special benefits.

## Overview

Vote for Paradise Bot on various listing sites to earn money, build voting streaks, and unlock exclusive bonuses that benefit your entire bot experience.

### Voting Benefits
- **Daily Rewards** - Earn millions of dollars
- **Streak Bonuses** - Exponentially increasing rewards
- **Farming Acceleration** - Crops grow 1.5 hours faster
- **Cooldown Reduction** - Rob command cooldown halved
- **Heist Bonuses** - Special multipliers for heist participation
- **Scratch Cards** - Bonus lottery tickets for voters
- **Property Protection** - Reduced property taxes

## How to Vote

### `/vote` Command
Opens the comprehensive voting interface with:
- **Real-time Status** - See which sites are available
- **Direct Links** - One-click access to voting pages
- **Reward Preview** - Next reward amount display
- **Streak Progress** - Current and highest streak tracking

### Available Voting Sites
- **Top.gg** - Primary Discord bot listing site
- **Discord Bot List (DBL)** - Secondary listing platform
- **Rank.top** - Additional voting opportunity

Each site has a 12-hour cooldown between votes.
