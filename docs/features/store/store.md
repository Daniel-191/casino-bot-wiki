# /store

Quick link to the external Paradise Bot Patreon store and Discord server.

## Description

`/store` doesn't sell anything in-game - it's a help embed that links you to the **external** purchase pages:

- **[Patreon](https://patreon.com/ParadiseBotx)** - real-money tier subscriptions for permanent perks.
- **[Discord](https://discord.gg/vMcbru7qMV)** - the support/community server.

For **in-game** items (bank tier upgrades), see [`/shop`](shop.md). For your **current tier dashboard**, see [`/patreon`](patreon.md).

## Usage

`/store`

## Examples

`/store`

## What you'll see

- Embed titled "✨ Explore Our Online Store!"
- Two buttons:
  - **Visit the Store** → opens patreon.com/ParadiseBotx in your browser
  - **Discord** → opens the support server invite

## Notes

- **Cooldown:** 3 seconds.
- This command does **not** purchase anything - it's a redirect only.
- For real in-game purchases (bank vaults), use [`/shop`](shop.md).
- For Patreon tier perks breakdown, see [Patreon Shop](patreon.md).
- For your current Patreon status, see [`/patreon`](patreon.md).
