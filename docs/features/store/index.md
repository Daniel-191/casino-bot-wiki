# Store & Items

Two stores in Paradise Bot - an in-game **bank-vault upgrade shop** ([`/shop`](shop.md)) and an external **Patreon store link** ([`/store`](store.md)). Plus the [`/patreon`](patreon.md) dashboard for checking your tier and benefits.

## Quick reference

| Command | What it does |
|---|---|
| [`/shop`](shop.md) | Buy bank vault tier upgrades with in-game money |
| [`/store`](store.md) | Link to the external Patreon store |
| [`/patreon`](patreon.md) | View your Patreon tier dashboard |

## Patreon (real money)

Real-money tier subscriptions via [Patreon](https://patreon.com/ParadiseBotx). 8 tiers from DONATOR to GODLY. Perks include:

- **+15% to +200% farming profit boost**
- **+10% to +66% farming speed**
- **+25% mining bonus + Mine All button**
- **0% heist tax** (Tier 6+)
- **15 lottery tickets/draw** vs 10 for free users
- **10–15 free scratch cards/week** (T1–T7 = 10, T8 = 15)
- **Larger heist daily limits**
- **Stronger bank yields**
- **Discord role flair + colored name**

See [Patreon Shop](patreon.md) for the tier-by-tier breakdown.

## Notes

- `/shop` cooldown: 3 seconds.
- `/store` is a link to the external Patreon page - no in-game purchases happen there.
- For Patreon gifts (won via scratch cards or admin gifting), see [`/scratch open`](../../economy/scratch/index.md) and `/stored-patreon`.
