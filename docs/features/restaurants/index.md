# Restaurants

An interactive minigame where you run fry deliveries with the Fries shop. Pay scales with weather, customer types, and special events with multiplier boosts.

## Commands

| Command | What it does |
|---|---|
| [`/fries-in-bag`](fries.md) | Start a delivery shift |
| `/fries-profile` | View level, XP, prestige, and unlocks |
| `/delivery-stats` | View your delivery stats |
| `/fries-event-status` | Check current global event |

Fries has a **level/XP system** (1–200) with a **prestige** option after capping. Each prestige adds a permanent earnings bonus.

## Shared systems

- **Weather conditions** - random per order, can boost or reduce pay.
- **Customer types** - different tip multipliers per customer.
- **Special events** - random global events with **2× to 8×+ pay multipliers** for a few hours.
- **Stats tracking** - `/delivery-stats` tracks your full history.

## Notes

- `/fries-in-bag` has a **4-second cooldown** - you can grind it quickly.
- Mistakes cost you money - wrong area or house = penalty.
- **Active events** show on the intro embed; pay attention before starting an order.
- For solo earning alternatives, see [Pets](../pets/index.md), [Fishing](../fishing/index.md), or [Mining](../mining/index.md).
