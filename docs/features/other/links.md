# /links

All Paradise Bot links in one place - Discord, Github, voting sites, Patreon, donation, bot invite.

## Description

A buttoned hub for everything external. Gives you one-click access to the support server, Github repo, voting pages, Patreon, PayPal donations, and the bot's invite link.

## Usage

`/links`

## Examples
