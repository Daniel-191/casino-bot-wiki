# /freemoney

A consolidated list of every command in the bot that earns money (free or otherwise) - daily, weekly, jobs, gambling, and more.

## Description

When you're new (or broke), this command tells you exactly what to run to make money. Lists every income command currently registered in the bot, with clickable command links.

## Usage

`/freemoney`

## Examples

`/freemoney`

## What it shows

Earning methods (clickable command mentions):

- **/daily** - daily reward
- **/weekly** - weekly reward
- **/monthly** - monthly reward
- **/vote** - vote rewards (Top.gg, DBL)
- **/beg** - random small payouts
- **/mine dig** - mining
- **/fish** - fishing
- **/farming** - crops
- **/hourly** - hourly reward
- **/crime** - risky reward
- **/lottery** - daily drawing
- **/properties** - passive income
- **/fries-in-bag** - delivery minigame
- **/pets** - pet work
- **/scratch open** - scratch cards from voting
- **/games** - gambling games

## Notes

- **Cooldown:** 3 seconds.
- Only includes commands the bot has currently loaded - newly added commands appear automatically.
- For one-shot vs recurring, see specific sections in **Earning Money**.
