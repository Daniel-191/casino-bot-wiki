# /quotes

Get a random gambling-themed quote. Pure flavor - no gameplay effect.

## Description

Pulls a random entry from a curated list of gambling quotes and prints it as an embed. Use when you need motivation, a laugh, or a reality check before betting your wallet on the next coinflip.

## Usage

`/quotes`

## Examples

`/quotes`

## Notes

- **Cooldown:** 3 seconds.
- No money or gameplay effect - purely cosmetic.
- Run it as many times as you like (within cooldown) for different quotes.
