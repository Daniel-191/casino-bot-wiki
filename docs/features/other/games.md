# /games

Browse every game in Paradise Bot, organized by category.

## Description

Lists every game registered in the bot, grouped into Casino / Adventure / Multiplayer categories with clickable command links. Useful when you forget what a game's exact slash command name is.

## Usage

`/games`

## Examples

`/games`
