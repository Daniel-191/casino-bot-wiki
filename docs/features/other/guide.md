# /guide

Interactive new-player guide. Walks you through the bot's core systems with buttons and embedded help topics.

## Description

The friendly entry point for new users. Opens an interactive embed with topic buttons covering everything from "How to make money" to specific feature deep-dives. New users get a special tutorial banner offering an interactive walk-through.

## Usage

`/guide`

## Examples

`/guide`

## What it covers

The guide includes topic shortcuts for:

- **Getting started** - accept TOS, daily commands, basic economy
- **Earning money** - daily/weekly/vote/jobs
- **Gambling games** - quick rules summary
- **Pets / Mining / Fishing / Farming** - feature explanations
- **Banking & properties** - money management
- **Patreon & donations** - supporter perks

## New user mode

If you've never used the guide before, the embed adds an **⭐ NEW USER SPECIAL ⭐** banner that recommends the **Interactive Tutorial** - a step-by-step walkthrough perfect for first-time players.

## Notes

- No cooldown specified - generally responsive.
- The bot remembers when you first used the guide and stops showing the new-user banner after that.
- For a flat list of free-money commands, see [`/freemoney`](freemoney.md).
- For a list of games, see [`/games`](games.md).
