# Other Features

Catch-all section for utility commands, support links, and small systems that don't fit elsewhere.

## Quick reference

| Command | What it does |
|---|---|
| [`/donate`](donate.md) | Link to PayPal donation page |
| [`/freemoney`](freemoney.md) | Lists every command that earns free money |
| [`/games`](games.md) | Browse all available games by category |
| [`/links`](links.md) | All Paradise-related links (Discord, Github, voting, etc.) |
| [`/quotes`](quotes.md) | Random gambling quote |
| [`/tos`](tos.md) | Accept the Terms of Service |
| `/verify` | Quick captcha verification (anti-selfbot) |
| [`/guide`](guide.md) | Interactive beginner's guide to the bot |

## Notes

- Most of these are 3-second cooldown helper/info commands.
- For real gameplay, see the **Games**, **Money Management**, **Earning Money**, and feature sections in the sidebar.
