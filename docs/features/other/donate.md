# /donate

A link to the PayPal donation page for supporting Paradise Bot's development.

## Description

Shows a friendly thank-you embed with two buttons: **Donate** (PayPal) and **Visit Our Store** (Patreon). Donations help cover hosting and development costs.

## Usage

`/donate`

## Examples

`/donate`

## Buttons

- [**Donate🥺**](https://www.paypal.com/donate/?hosted_button_id=W7KDRFDEETEFU) → opens PayPal at the developer's hosted donation form.
- [**Visit Our Store!**](https://patreon.com/ParadiseBotx) → opens patreon.com/ParadiseBotx for tier subscriptions.

## Notes

- **Cooldown:** 3 seconds.
- This command **does not** spend in-game money - it links to real-money donations only.
- For tier subscriptions with permanent in-game perks, see [Patreon Shop](../store/patreon.md).
- For the Discord/Github/voting links, use [`/links`](links.md).
